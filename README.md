Установка:
==================

Необходимо установить webpack
```
npm install --global webpack
```

Установить окружение из package.json
```
npm install
```

Ставим typescript и ts-node
```
npm i -g typescript
npm i -g ts-node
```

Запуск сборки:
==================
```
 webpack
```
